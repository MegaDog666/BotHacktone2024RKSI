----------------------------------------------------------------ИНСТРУКЦИЯ------------------------------------------------------------------------
1. Получение доступа к боту:
1.1 Для получения доступа к боту требуется в файле "info.py" в константе "TOKEN" вставить свой  токен.
1.2 Теперь, когда бот управляется вами, он будет находиться на сервере, который будет поддерживать бота все время в рабочем состоянии


2. Использование, будучи пользователем:
2.1 Заходим в чат с ботом
2.2 Нажимаем кнопку "Start"
2.3 Выбираем предпочтения по еде
2.4 Выбираем предпочтения по интересам
2.5 Подтверждаем выбор или его редактируем в случае желания изменить выбор
2.6 Отправка местоположения и подтверждение отправки данных
2.7 Выбор функции бота
	2.7.1 Что поесть?
	2.7.2 Где поесть?
	2.7.3 Совет от Chat GPT
2.8 Получение ответа на выбранную категорию


-----Для установки всех зависимостей в боте используется файл requirements.txt-----

pip install -r requirements.txt - команда для установки зависимостей(прописывать внутри проекта)

----Как работает сервер:
На сервере есть база данных и сам бот. Через Docker-compose база данных и сам бот взаимодействуют, запускаются в одной экосистеме Docker'а. 
Преимущества данного подхода:
 1.Сервер может работать независимо от разработчика
 2.Бот работает тогда, когда работает сервер 
 3.Высокая безопасность бота и защита базы данных
 4.Оптимизация нагрузки 
 5.Docker-compose также позволяет быстро и из любой точки мира администрировать бота
